# Do You Love Me? ❤️

A playful interactive web project built with **HTML, CSS, and JavaScript**.  

This small website asks the user:  
> “Do you love me?”  

It features:  
- **Yes / No buttons**:  
  - Clicking **Yes** advances the dialogue to “Are you sure?” and then “How much?”, finally allowing the user to type a response.  
  - Clicking **No** makes the button **run away**, shrink, and get redder, creating a fun “angry button” effect.  
- **Exploding No button**: After hovering over the No button enough times, it triggers a mini explosion animation with a cheeky message.  
- **Confetti animation**: When the user finally types their answer, confetti celebrates the response.  
- Fully **interactive and playful UI**, centered text and buttons, all animations handled with JavaScript.
